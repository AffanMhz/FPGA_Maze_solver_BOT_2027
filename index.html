Explore Project

Project Overview

This project focuses on designing and simulating an FPGA-based maze-solving robot using Verilog HDL. The robot navigates through a maze autonomously using sensor inputs and control algorithms implemented in hardware.

Verilog HDL

Hardware description language for designing digital circuits and implementing control logic.

Intel Quartus

FPGA development and synthesis tool for compiling and programming Verilog designs.

ModelSim

Advanced simulation environment for verifying and testing digital designs before implementation.

GitHub Collaboration

Version control and team collaboration for managing project files and documentation.

Project Tasks

Explore the detailed breakdown of our project components and implementation phases.

Task 1: PWM Motor Control

+

Objective: Implement Pulse Width Modulation for precise motor speed control.

Key Components:

  • PWM signal generator module
  • Duty cycle control logic
  • Motor driver interface
  • Speed regulation algorithm

Implementation: Designed a configurable PWM module with adjustable frequency and duty cycle for smooth motor control.

Task 2: Sensor Integration

+

Objective: Interface ultrasonic and IR sensors for obstacle detection.

Key Components:

  • Ultrasonic sensor driver
  • Distance calculation module
  • IR sensor array processing
  • Sensor fusion algorithm

Status: Successfully implemented multi-sensor data acquisition and processing pipeline.

Task 3: Maze Navigation Algorithm

+

Objective: Develop and implement wall-following and pathfinding algorithms.

Algorithms Implemented:

  • Left-hand wall following
  • Right-hand wall following
  • Dead-end detection and backtracking
  • Path optimization

Performance: Achieved 95% maze completion rate in simulation tests.

Task 4: State Machine Design

+

Objective: Create a robust finite state machine for robot control.

States:

  • IDLE - Initial state
  • MOVE_FORWARD - Forward motion
  • TURN_LEFT - Left rotation
  • TURN_RIGHT - Right rotation
  • OBSTACLE_DETECTED - Decision making
  • COMPLETE - Goal reached

Task 5: Simulation & Testing

+

Objective: Comprehensive testing using ModelSim and testbenches.

Testing Phases:

  • Unit testing of individual modules
  • Integration testing of complete system
  • Timing analysis and optimization
  • Waveform verification

Results: All modules passed functional verification with timing constraints met.
